# 宝藏网址收藏 - 图片资源

这个仓库专门存放 gudaoqihuo.com 的图片资源，避免占用主站仓库空间。

## 目录结构

- `ads/` - 推广广告图片
- `resources/` - 其他资源图片

## 引用方式

在主站 HTML 中使用以下格式引用图片：

```html
<img src="https://raw.githubusercontent.com/xiaokangjiao-ai/gudaoqihuo-assets/main/ads/图片文件名.jpg" alt="描述">
```

或者使用 GitHub Pages 链接（需要开启 Pages）：

```html
<img src="https://xiaokangjiao-ai.github.io/gudaoqihuo-assets/ads/图片文件名.jpg" alt="描述">
```
